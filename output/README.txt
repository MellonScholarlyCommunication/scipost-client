SciPost Generated Events
-=-=-=-=-=-=-=-=-=-=-=-=

Submissions types:

arxiv.org - SciPost submissions from arXiv org
doi.org -  SciPost submissions from a DOI (no other source given)
scipost.org/preprint - SciPost submissions from own SciPost preprint service

Responses:

scipost.org/submissions - Outgoing Accept events for any submissions
scipost.org/publications - Outgoing Announce events for publications
